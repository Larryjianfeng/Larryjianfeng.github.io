FastText分类器

The method

The link to the github: fastText

The code to use it is pretty simple.

					# for trainning the data
					./fasttext supervised -input input_data_path_here -output output_model_path_here
					# for making prediction
					./fasttext predict model_path_here data_path_to_predict_here 1
					# for testing
					./fasttext test model_path_here data_path_to_predict_here 1

fasttext为c++编写，不过钱哥已经编译出python库：https://github.com/xi-qian/fastText
